# airails

Development conventions, [skills](https://agentskills.io/specification), and [guidelines](https://agents.md) for building modern Java and web applications.

## What's Inside

- **javacli** — Guidelines for Java 25 CLI applications using source-file mode (JEP 458), no build tools required
- **microprofile** — Conventions for Java 25 + MicroProfile/Jakarta EE projects following the BCE (Boundary-Control-Entity) pattern
- **static-web-development** — Standards for modern static websites using semantic HTML5 and CSS without external dependencies
- **microprofile configuration** — Java-specific configuration extending the microprofile skill
